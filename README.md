 - Flutter SDK 받기 하단에 알집을 다운받는다.
 - 다운받는 중간에 C 드라이브에 flutter 를 설치할 폴더를 하나 만들어준다. ( ex : C:\dev )
 - 폴더만든곳에 다운받은 알집을 푼다
